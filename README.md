# Desktop Clock (Python)

A lightweight desktop clock built using Python and Tkinter.  
The clock uses a transparent background and minimal UI, allowing it to blend cleanly with the desktop without distractions.

## Features
- Real-time digital clock (HH:MM:SS AM/PM)
- Transparent background
- Borderless window
- Automatically positions itself at the top-center of the screen
- Customizable display name and colors

## Requirements
- Python 3.x
- Tkinter (comes pre-installed with most Python distributions)

## How to Run
Clone the repository and run the script:

```bash
python clock.py
